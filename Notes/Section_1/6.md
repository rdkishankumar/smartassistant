# Notes on Setting Up OpenAI API Key and Testing Spring AI Application

## OpenAI vs. ChatGPT
- OpenAI: The company focusing on building AI models (like Apple).
- ChatGPT: One of its products, an intelligent chatbot using OpenAI's LLM models (like iPhone).

## Setting Up OpenAI Account and API Key
- Go to [platform.openai.com](https://platform.openai.com).
- Log in (use same account as ChatGPT if applicable).
- Go to Settings > Billing.
- Add at least $5 to your balance (via credit card; works internationally).
    - $5 covers the entire course demos.
    - Alternatives if not adding funds:
        - Set up local LLMs (less powerful but testable).
        - Watch demos and test later.
        - Share API key with friends (one account suffices).
- Go to API Keys section.
- Create a new secret key (starts with "sk-").
    - Give it a name and select a project (default or custom).
    - Copy and store securely.

## Pricing Information
- Charges per 1 million tokens (input + output size).
- Examples:
    - GPT-4o-mini: Cheaper for smaller models.
    - GPT-4o or preview models: More expensive.
- Models evolve; concepts remain the same.
- Spring AI allows selecting models (covered later).

## Configuring API Key in Spring AI App
- In `application.properties`, add: `spring.ai.openai.api-key=${OPENAI_API_KEY}`
    - Uses environment variable to avoid hardcoding.
- Set environment variable:
    - Name: `OPENAI_API_KEY`
    - Value: Your copied API key.
- In IntelliJ: Run > Edit Configurations > Environment Variables > Add.

## Running and Testing the Application
- Build and start: Should run without errors (bean creation succeeds).
- Test API using Postman (or browser).
    - Method: GET
    - URL: `http://localhost:8080/api/chat?message=What is your name and which model you are using?`
    - Response: e.g., "I'm ChatGPT, based on OpenAI's GPT-3.5 model." (May vary by Spring AI version/model).
- Try another prompt: `Tell me a joke about Spring Boot`
    - Example response: "Why did the Spring Boot application break up with its database? Because it couldn't handle the commit issues."
    - Note: LLMs are non-deterministic (probabilistic), so responses vary for a human-like feel.

## Behind the Scenes
- Spring AI handles OpenAI endpoints and complexities (e.g., in `OpenAiChatModel.internalCall()`).
- API endpoint: See `OpenAiApiConstants.DEFAULT_BASE_URL` for details.
- Focus on business logic; framework abstracts vendor specifics.
