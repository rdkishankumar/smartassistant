# Notes on Spring AI

## Introduction
Imagine a Spring Boot application that suddenly becomes intelligent—not just fetching data, but providing answers, writing emails, and automating redundant tasks. Welcome to the world of Spring AI, which integrates AI capabilities (like LLMs such as ChatGPT, Ollama, DeepSeek) into Spring Boot applications. It's a bridge between AI models and Java developers, bringing AI power into Java hands without switching languages or learning complex SDKs.

## What is Spring AI?
- Spring AI is a new project in the Spring ecosystem.
- It enables invoking LLMs via simple REST API logic, but provides more.
- Key benefit: If you know Spring Boot, you're halfway to integrating AI.

## Why Spring AI Over Manual Integration?
- **Multi-Provider Support**: Handles various LLM vendors (e.g., tens of providers with different endpoints, security standards). Spring AI abstracts boilerplate, allowing easy switching between models/vendors.
- Focus on business logic; framework handles complexities.

## Key Features
- **MCP Protocol Support**: In the GenAI world, MCP (like HTTP in web) enables building MCP client and server applications easily.
- **Vector Store Integration**: Store organizational documents/structured data in vector databases (e.g., Pinecone, Redis, Postgres, MongoDB) for LLM access. Essential for scenarios beyond simple invocations.
- **Advanced Patterns**:
    - Chat memory
    - Tool/function calling
    - Advisor APIs for reusable AI logic
- **DevOps Features**:
    - Observability: Monitor AI operations.
    - Guardrails: Evaluate outputs for reliability.
- **Spring Ecosystem Integration**:
    - Auto-configuration and starter projects for quick setup with AI models and vector DBs.
    - Familiar concepts like Spring Dependency Injection.

## Typical Use Cases
- AI-powered customer support (automate support activities).
- Natural language document search.
- Personalized recommendations.
- Data processing.
- Automating redundant tasks.

## Conclusion
Spring AI goes beyond invoking LLMs by providing a surrounding ecosystem for common AI patterns. As a Spring/Spring Boot developer, you'll find it familiar and powerful. Throughout the course, we'll explore these features in depth.
